Spacing Utilities - Margin and Padding

{property}{sides}-{breakpoint}-{size} Ex: pl-sm-1

{property}{sides}-{size} If we omit the {breakpoint} then, it is effective on all the devices. Ex: pl-1

{property}-{size} - If we omit the {sides}, then property will be applied to all sides. Ex: p-1

Below are possible values

{property} - m, p

{sides} - t, b, l, r, x, y

{breakpoint} - sm, md, lg, xl

{size} - 0, 1, 2, 3, 4, 5, auto

{size} ranging from 0-3 rem. 1rem = 16px. 'auto' works only for margin.
